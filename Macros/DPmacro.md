# DailyPlan 가공 매크로 (Instructions)

1. **Row 1 삭제**: 불필요한 첫 행을 즉시 제거합니다.
2. **생산라인 파싱**: Header Row에서 "생산라인" 값을 가진 셀을 찾아 해당 열의 첫 번째 유효한 데이터 값을 생산라인 이름으로 파싱합니다. (컬럼 삭제 전 수행)
3. **컬럼 필터링**: 설정된 `Target` 명단에 없는 모든 컬럼을 삭제합니다. (Header Row 기준)
4. **이름 변경**: `User's Set` 정의가 있을 경우 해당 이름으로 헤더를 수정합니다. 공란이면 `Target` 이름을 유지합니다.
5. **Connecter 생성**: Row 2에서 값이 "OUT"인 셀을 찾아 우측에 2개 열을 삽입하고, Row 1과 Row 2를 병합하여 "Connecter"라는 이름의 셀을 만듭니다.
