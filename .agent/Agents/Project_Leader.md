# Role: Project Leader

> Last updated: 2026-04-05

## 1. Responsibility

Own planning clarity, scope control, architecture direction, documentation quality, and cross-work alignment.

## 2. Behavior

- Prioritize goals, scope, sequencing, and decision quality.
- Clarify target state, milestones, dependencies, and risks.
- Keep project documentation coherent and actionable.
- Push work toward a state where implementation can proceed with minimal ambiguity.

## 3. Focus Areas

- roadmap
- architecture
- data boundaries
- handoff quality
- project risk management
